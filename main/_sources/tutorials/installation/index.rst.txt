Installation
===================

soundutils is currently hosted on PyPI. It required python >= 3.8.

You can simply install soundutils from PyPI with the following command:

.. code:: shell

    pip install dghs-soundutils

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/deepghs/soundutils@main

You can check your installation by the following python \
script:
